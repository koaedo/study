## 🎯 주요 학습 내용

### 🔄 Transform (변환)

1. **3D 좌표계**
   - X, Y, Z 축의 이해
   - 오른손 좌표계
   - AxesHelper로 시각화

2. **Position (위치)**
   - 3D 공간 상의 위치 설정
   - `position.set()`, `length()`, `distanceTo()` 메서드
   - 상대적 이동과 애니메이션

3. **Scale (크기)**
   - 배율 기반 크기 조절
   - 각 축별 독립 조절
   - 호흡 효과, 펄스 효과 구현

4. **Rotation (회전)**
   - 라디안 개념과 각도 변환
   - 회전 순서 (Rotation Order)
   - 연속 회전과 부드러운 회전

5. **dat.GUI**
   - 실시간 값 조절 도구
   - 폴더로 그룹화
   - 다양한 컨트롤 타입 활용

## 🚀 사용 방법

### 인터랙티브 발표 슬라이드 (강력 추천!) ⭐

1. **파일 열기**
   - `Transform_인터랙티브_슬라이드.html` 파일을 웹 브라우저로 엽니다

2. **네비게이션**
   - **키보드**: ← → 화살표 키로 슬라이드 이동
   - **마우스**: 좌우 버튼 클릭
   - **모바일**: 화면을 좌우로 스와이프
   - **하단 점**: 특정 슬라이드로 바로 이동

3. **라이브 데모 조작**
   - 각 슬라이드 오른쪽에 실제 Three.js 씬이 표시됩니다
   - 버튼을 클릭하여 실시간으로 객체를 조작할 수 있습니다
   - **Position 슬라이드**: 화살표 버튼으로 큐브 이동
   - **Scale 슬라이드**: 각 축 크기 조절 및 전체 크기 조절
   - **Rotation 슬라이드**: 각도 단위로 회전, 자동 회전 토글
   - **종합 슬라이드**: 프리셋과 애니메이션 효과

3. **발표 효과 극대화**
   - 청중의 주의 집중
   - 지루하지 않은 발표

4. **반복 학습**
   - 원하는 만큼 버튼을 눌러 실험
