# Three.js 학습 정리 발표자료

## 📋 개요
Three.js를 학습하면서 깨달은 점들을 정리한 발표자료입니다.

## 📂 파일 구성
- `Three.js_학습정리_슬라이드.html` - 슬라이드 형식 발표자료 (추천!)
- `Three.js_학습정리.html` - 스크롤 형식 발표자료
- `README.md` - 이 파일

## 🎯 주요 내용

### 1. 3D 씬의 기본 구성 요소
- Geometry, Material, Mesh의 관계
- Camera와 Light의 역할
- Renderer의 기능

### 2. 카메라의 시야 범위 (Near & Far)
- Near와 Far의 개념
- 실제 카메라와의 유사성
- 성능 최적화와의 관계

### 3. 카메라의 종류
- Perspective Camera (원근 카메라)
- Orthographic Camera (정사영 카메라)
- 각 카메라의 사용 사례

### 4. Geometry와 Material의 공유
- 효율적인 메모리 관리
- 게임 아이템 복사 개념
- 성능 최적화 방법

## 🚀 사용 방법

### 슬라이드 버전 (추천)
1. `Three.js_학습정리_슬라이드.html` 파일을 웹 브라우저로 엽니다
2. 화살표 키(← →) 또는 화면의 버튼을 눌러 슬라이드를 이동합니다
3. 하단의 점(•)을 클릭해서 특정 슬라이드로 바로 이동할 수 있습니다
4. 모바일에서는 스와이프로 슬라이드 이동 가능합니다

### 스크롤 버전
1. `Three.js_학습정리.html` 파일을 웹 브라우저로 엽니다
2. 스크롤하며 각 섹션을 확인합니다
